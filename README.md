# 📌 Project Overview

This project focuses on performing a complete Quality Assurance process for the **Swag E-Commerce** platform, covering both the **Web Application** and the **Mobile Application**.

The project was completed by the QA team **Validatx** as part of the **Digital Egypt Pioneers Initiative (DEPI)** in collaboration with **ITC**.

The testing process included preparing all testing documentation, executing manual testing, API testing, identifying defects, and developing automation scripts using Selenium.

---

# 🎯 Objectives

- Ensure the quality of the application.
- Verify that all requirements are implemented correctly.
- Detect and report software defects.
- Improve the overall user experience.
- Validate business requirements.
- Perform comprehensive testing for both Web and Mobile applications.
- Build automation scripts for repetitive test scenarios.
